search ( string, mode ) dropkick.js:630

Finds all options who's text matches a pattern (strict, partial, or fuzzy)

"strict" - The search string matches exactly from the beginning of the option's text value (case insensitive).

"partial" - The search string matches part of the option's text value (case insensitive).

"fuzzy" - The search string matches the characters in the given order (not exclusively). The strongest match is selected first. (case insensitive).

Parameters:
  • string <String>

    The string to search for

  • mode <Integer>

    How to search; "strict", "partial", or "fuzzy"


Returns: <Boolean>

An Array of matched elements
